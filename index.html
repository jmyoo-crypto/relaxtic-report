WHY — 현상
자사몰 매출이 전전주보다 203.8% 급증했다
지난주 10,644,211원 / 242건  ←  전전주 3,503,312원 / 71건.
메타 광고비는 -2.8%로 거의 동일했음에도 자사몰 매출은 3배 이상 급증했다.
🤔
WHAT — 질문
광고비가 동일한데 왜 자사몰 매출은 3배가 됐을까?
  • 소재 효율이 크게 개선됐다 (CTR 1.82% → 2.53%).
  • 전환율 또는 객단가가 상승했을 가능성이 있다.
  • 지난주 화요일 거누파파네 콘텐츠 라이브 진행 → 자사몰 유입 집중.
  • 지난주 금요일 깐돌네 콘텐츠 라이브 진행 → 추가 매출 견인.
💡
HYPOTHESIS — 추가 가설
콘텐츠 라이브 2회 + 소재 효율 개선 + 프로모션 병행 효과
  • 콘텐츠 라이브 직접 기여 — 화요일 거누파파네 / 금요일 깐돌네 라이브가 자사몰 트래픽을 집중 견인. 메타 광고가 리타겟팅 역할을 하며 라이브 시청자의 구매 전환을 높였을 가능성.
  • 77% 할인 소재 전환 폭발 — CPA 45,073원 → 11,257원으로 대폭 개선. 예산도 2배 증액(135,220 → 270,173원).
  • 타이밍은지금 소재 효율 개선 — CPA 16,794 → 8,425원 ▼49.8%. 광고비 감소에도 구매수 증가(17 → 27건).
  • 신규 소재 타이밍_일반_1 투입 — 전전주 0원 → 지난주 207,963원. CPA 14,854원으로 첫 주 양호.
